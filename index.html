Now, what is Spark.js

a lightweight 3d particle engine in js

  • Slim core
  • Lots of plugins: you can do your own
  • Architecture: stack based
    • One stack to create particles
    • One stack to change particles during their life time

More on sparks.js

  • Author: zz85 / @BlurSpline / blog
  • License: MIT
  • Source: https://github.com/zz85/sparks.js
  • Made for three.js and tween.js
    • But work on others 3D engines too

What is Spark.js Editor

Motivations

  • Inpired from:
    • glsl editor by mrdoob / Gallery
    • shadertoy by Inigo Quilez
    • recently css editor dabblet by lea verou
  • Needed for game 3D effects
    • Games effects are like salt in pasta
    • Hard to design without editor

An Live Online editor

  • Live editor rocks
    • immediat result. no latency
    • Very agile way to design.
  • Online is cool
    • bookmarkability: so all states in the url
    • long ugly url ? url shortening to the rescue
  • All static files
    • Easy to host
    • Free to host
